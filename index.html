Jogo da Cobrinha

Este projeto, iniciado em meados de 2024, foi um dos meus primeiros passos no aprendizado de lógica de programação. Inspirado no clássico 'Jogo da Cobrinha' de 1988, desenvolvi a aplicação completamente em Python, utilizando a biblioteca Pygame. O projeto representa uma fusão de conceitos básicos de programação com o uso de bibliotecas gráficas para criar um jogo funcional e interativo.

Los Pedro's Customs

Este projeto foi desenvolvido em setembro de 2024, em colaboração com colegas de sala, como parte de uma série de desafios para aprender Javascript. Criamos um site para a apresentação de uma oficina mecânica, com uma ferramenta de orçamentos. Utilizamos apenas HTML, CSS e Javascript. O primeiro desafio foi implementar um sistema para selecionar marca, modelo e ano de veículos, o que nos levou a integrar a API da Tabela FIPE para buscar os dados necessários de forma dinâmica. O segundo desafio foi tornar o site completamente responsivo, garantindo uma boa experiência em qualquer dispositivo, e fomos bem-sucedidos em ambos.

Industria Feliz

Entre abril e junho de 2024, desenvolvi meu primeiro grande projeto na área de desenvolvimento web, enfrentando o maior desafio da minha jornada até o momento. Utilizei Python com o framework Flask, além de HTML, CSS e MySQL, para criar uma aplicação destinada a indústrias. A ferramenta permite o registro de materiais e preços, gerando relatórios detalhados de custos empresariais. Esse projeto foi uma excelente oportunidade para entender a complexidade de construir uma aplicação completa do zero, desde o backend até o frontend, e marcou um avanço importante no meu aprendizado como programador.
