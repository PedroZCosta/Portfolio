Jogo da Cobrinha

This project, started in mid-2024, was one of my first steps in learning programming logic. Inspired by the classic 'Snake Game' from 1988, I developed the application entirely in Python, using the Pygame library. The project represents a fusion of basic programming concepts with the use of graphical libraries to create a functional and interactive game.

Los Pedro's Customs

This project was developed in September 2024, in collaboration with classmates, as part of a series of challenges to learn Javascript. We created a website for the presentation of an auto repair shop, with a budget tool. We used only HTML, CSS, and Javascript. The first challenge was to implement a system to select the vehicle's brand, model, and year, which led us to integrate the FIPE Table API to fetch the necessary data dynamically. The second challenge was to make the website fully responsive, ensuring a good experience on any device, and we were successful in both.

Industria Feliz

Between April and June 2024, I developed my first major project in web development, facing the biggest challenge of my journey so far. I used Python with the Flask framework, along with HTML, CSS, and MySQL, to create an application aimed at industries. The tool allows the registration of materials and prices, generating detailed business cost reports. This project was an excellent opportunity to understand the complexity of building a complete application from scratch, from backend to frontend, and marked an important milestone in my learning as a programmer.
